# Josh - 多功能工具集

一个为开发者设计的实用工具集合，包含各种提高工作效率的工具应用。

## 🛠️ 可用工具

### [Markdown Editor](/apps/markdown-editor/)
- **主要作用**: 强大的Markdown编辑器，支持实时预览和微信公众号排版优化

## 🚀 快速开始

1. 克隆项目到本地
2. 打开 `index.html` 访问工具集主页
3. 选择需要的工具开始使用

## 📁 项目结构

```
josh/
├── apps/                 # 工具应用目录
│   └── markdown-editor/  # Markdown编辑器
├── index.html            # 项目入口页面
└── README.md            # 项目说明文档
```

## 🔧 开发

每个工具都是独立的Web应用，可以直接在浏览器中运行。如需本地开发服务器，可以进入对应工具目录运行：

```bash
cd apps/markdown-editor
node server.js
```
