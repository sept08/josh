# Markdown Editor

一个功能强大的Markdown编辑器，专为微信公众号排版优化设计，支持实时预览和富文本复制功能。

## ✨ 核心功能

### 实时预览
- 左侧编辑，右侧实时预览
- 支持标准的Markdown语法
- 代码高亮显示

### 微信公众号排版优化
- **自动美化样式**：行间距1.6、字体颜色#3f3f3f、字号16px
- **标题序号**：一级标题自动添加大序号（如01、02），序号颜色为主题色
- **强调样式**：加粗、斜体文本显示为主题色(#35B378)
- **链接样式**：链接颜色统一为主题色，无下划线
- **引用块优化**：浅绿色背景(#f5f9f5)，左侧主题色边框，文字颜色与正文一致
- **代码块优化**：编辑器风格代码块，浅灰色背景(#f6f8fa)，深色文字(#24292e)
- **任务列表**：已勾选复选框显示为主题色，未勾选为浅灰色

### 富文本复制
- 一键复制美化后的富文本内容
- 完美适配微信公众号编辑器
- 保留所有样式和格式

### 任务列表支持
- 支持GitHub风格的复选框任务列表
- 自动优化任务列表显示样式
- 复制时保持勾选框与文本同行显示

## 🚀 使用方法

1. 在左侧编辑区输入Markdown内容
2. 右侧实时预览美化效果
3. 点击"复制富文本"按钮
4. 直接粘贴到微信公众号编辑器

## 🛠️ 技术特性

- **前端技术**: HTML5, CSS3, JavaScript (ES6+)
- **Markdown解析**: marked.js
- **样式处理**: 自定义CSS样式系统
- **复制功能**: Clipboard API + 富文本处理
